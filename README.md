# Wiki Search

A simple search engine that leverage the Wikipedia API. Made with Vue3

## Demo URL

https://yz-wiki-search.netlify.app

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
